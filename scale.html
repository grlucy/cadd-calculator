Note about rounding:

The Scale Down column is a decimal rounded to six decimal places. Due to rounding, there would be a slight difference between an original object and that object after it has been scaled down and then scaled back up. This should be avoided if at all possible.

If absolutely necessary to scale up a previously scaled down object, the number in the Scale Up (Actual) column will give you a much closer result; however, it still won't be perfect. You can also use this as a dimension scale factor for checking dimensions on a scaled-down object.
